## ðŸ” How to Use Each Feature

### Search Functionality

**Available on most pages:**
- Type keywords in the search box
- Works across multiple fields:
  - Rankings: Organization name, category
  - Awards: Award name, giver, category
  - Achievements: Title, description
  - Media: Publication name, headline
- Results update instantly

### Filtering

**Common filters:**
- **By Type:** National/International (Rankings)
- **By Category:** Dropdown selection
- **By Year:** Select specific year
- **By Impact:** Major/High/Medium (Achievements)

**How to use:**
1. Select filter option from dropdown
2. Results update automatically
3. Click "Reset Filters" to clear all

### Featured Content

**Special highlighting:**
- Orange borders indicate featured items
- Featured rankings show top positions
- Featured media mentions are highlighted
- Latest report is promoted on Reports page
